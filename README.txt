openSUSE
========

openSUSE 13.1

Como o nome já diz, é um programa de pos instalação do openSUSE 13.1, 
em que o usuário pode instalar todos os pacotes necessários para 
tornar o sistema utilizável no dia a dia.

Esse programa não tem muita utilidade para usuários mais avançados 
mas sim para usuários novatos que sairam do ubuntu ou de outra distribuição.


Programa de pos instalação do openSUSE 13.1 
Compatível com KDE e GNOME 
Versão 1.0
